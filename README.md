# 搜索数组平衡点的案例

给定一个整数数组，数组的“平衡位”是这样的位置： 它左边所有的数字之和===右边所有数字之和。

例如：

arr = [1，2，3，0，6]

这个数组中， 0所在的位置是“平衡位”， 因为左边数字之和 1+2+3=6

右边数字之和6=6

0所在的位置下标为3, 亦即， arr[3] === 0
